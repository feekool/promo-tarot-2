Запуск форума за 5 шагов

Даже если вы никогда не работали с кодом — вы справитесь. Пошаговая инструкция с примерами.

  1. 1

    Создайте аккаунт на GitHub

    Бесплатная регистрация на github.com займёт 2 минуты.

  2. 2

    Откройте шаблон Feekool Tarot

    Перейдите по ссылке: github.com/feekool/feekool.github.io

  3. 3

    Создайте свой репозиторий

    Нажмите "Use this template" → "Create a new repository". Назовите, например, my-tarot-space.

  4. 4

    Включите GitHub Pages

    Settings → Pages → Branch: main → Save. Ваш форум будет доступен через 1-2 минуты.

  5. 5

    Настройте под себя

    Добавьте разделы: "Расклады", "Курсы", "Витрина", "Отзывы". Измените цвета, логотип, тексты — всё в файлах шаблона.
